Understanding How this all works!

Microsoft Windows Security Update for July 2024 Applying update manually using the script below : Copy msu file over to server – checking the kb is correct for the server that have remoted onto. Open CMD with Elevated permissions – Run as Administrator Cd %homepath%\Desktop wusa.exe windows10.0-kb5040430-x64_9879d14c817a7aec56b0ae1f73daecc7a678b3aa.msu /quiet /norestart Each KB article is for each version of windows server : - KB5040430 KB5040437 KB5040434 These type of updates are being applied in LCS on the last servers that are not being automated. By running the above update with the commands in place stops the update from rebooting the server allowing the administrator to reboot at another time where possible. Stopping random reboots. In short supressing windows update.
