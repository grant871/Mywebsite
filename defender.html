Windows Defender Elevation of Privilege Vulnerability

Microsoft Defender is prone to Elevation of Privilege Vulnerability. Affected Software:Windows Defender Antimalware Platform

An attacker who successfully exploited this vulnerability could gain specific limited SYSTEM privileges.

Update Windows Defender :


  • Launch Windows Security through the Start menu and select Virus & threat protection.
  • Under Virus & threat protection updates, click on Protection updates.
  • Select Check for updates, and Windows Security will attempt to download the latest updates from Microsoft.
