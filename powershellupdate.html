Removing Folders from a file System

Remove-Item -Path "C:\Program Files\Video Lan" -Recurse -Force

The above script can be used to removed a folder from a windows file system. Why is there a focus for doing this - well if some or a an automated system has not removed a program correctly but a vulnerability detector stil detects the program this is one way we can ensure the .exe is not present to be exploited.
